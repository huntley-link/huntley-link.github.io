Every Day I Write The Book.

Github LinkedIn Instagram

This web site is an attempt to bring the software engineering paradigm of “Continuous Integration/Continuous Deployment” to to my personal work, be it technical, literary, epistolary, journaling, note-taking, brainstorming, indexing, archiving etc.

My intent is to bring together everything I’ve preserved of my past efforts and document the process of creating new stuff, in what used to be called “postmodern” fashion – showing my work and all the messy parts.

I aspire, via this site and the Quarto publishing platform, to organize my work into volumes, which can be released in ebook and print-ready formats in continuous editions updated with every iteration of the site. Each step of the authoring, editing, compiling and publishing process will be captured and laid bare for posterity in a Github repository.
